# Sakila Database SQL Örnekleri

Bu çalışmada **Sakila** veritabanı kullanılarak farklı SQL sorguları hazırlanmıştır.  
Örnekler, özellikle **GROUP BY**, **COUNT**, **MAX**, **MIN** **SUM** **AVG** ve **HAVING** komutlarının kullanımını pekiştirmek amacıyla yapılmıştır.  
Her sorgunun üstünde, bizden neyin istendiği belirtilmiştir.
